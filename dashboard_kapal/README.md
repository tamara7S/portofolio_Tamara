# 🚢 Dashboard Monitoring Armada Kapal

Dashboard web berbasis real-time untuk monitoring posisi dan status armada kapal PT. PAL Indonesia.

## Fitur
- 🗺️ **Peta Interaktif** — Visualisasi posisi armada kapal di seluruh perairan Indonesia menggunakan Leaflet.js
- 📊 **Statistik Armada** — Ringkasan jumlah kapal berdasarkan status (Operasional, Maintenance, Berlabuh)
- 📋 **Tabel Armada** — Daftar lengkap kapal dengan fitur pencarian dan filter status
- 📝 **Log Aktivitas** — Riwayat aktivitas terbaru armada secara real-time
- 🌙 **Dark Mode UI** — Tampilan profesional bertema kelautan

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- [Leaflet.js](https://leafletjs.com/) — Peta interaktif
- [Chart.js](https://www.chartjs.org/) — Visualisasi data donut chart
- OpenStreetMap — Tile layer peta gratis

## Cara Menjalankan
1. Clone repository ini
2. Buka file `index.html` di browser
3. Tidak memerlukan server atau instalasi tambahan
